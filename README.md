
该仓储是基于[NetModular](https://github.com/iamoldli/NetModular)框架开发的人事档案模块

### 本模块功能较简单，仅供学习参考，如若要在正式项目使用，请慎重~

# 启动方式

进入 UI 目录，执行以下命令

```
npm install #安装包
npm run build #打包
```

在 VS 中启动 WebHost 或者进入 WebHost 目录执行以下命令

```
dotnet watch run
```

打开 [http://localhost:6224](http://localhost:6224)，输入账户密码 admin/admin
